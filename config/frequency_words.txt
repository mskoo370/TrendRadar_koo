# 规则说明：
# 1. 每行一个词
# 2. 以 ! 开頭的词表示排除（如果标题包含该词，则不推送）
# 3. 以 + 开頭的词表示必须包含（如果要推送，标题必须包含该词）
# 4. 普通词表示包含（标题包含该词即可推送）
# 5. 支持 /正则表达式/ 语法
# 6. 支持 => 别名语法 (例如: /pattern/ => Alias)
